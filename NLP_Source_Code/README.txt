# CECZ4045 Natural Language Processing Group 7
# Project - Sentiment Analysis on Apple from Twitter Tweets based on Apple Event 2022

## About / Synopsis

* This project aims to conduct sentiment analysis base on Twitter tweets on the topic of Apple Event 2022
* Project status: completed

## Content

> * 1 Crawling
>   * Crawling of data from Twitter API
> * 2a Classification & Innovation - Vectorizers
>   * Initial classification attempt using various vectorizers + Innovated classifier
> * 2b Classification - LSTM
>   * Deep learning classification using LSTM
> * 2c Classification - BERT
>   * Deep learning classification using BERT transformer
> * 3a Classification & Innovation - Selective Stopwords
>   * Revision experimental classification by selective removal of stopwords + Innovated classifier
> * 3b Classification & Innovation - With Stopwords
>   * Revision experimental classification by not removing stopwords + Innovated classifier
> * 4 Innovation + Simple UI
>   * Innovated classifier integrated in Streamlit UI

## Installations

* Jupyter Notebook in Anaconda environment (Recommended)
* Google Colab
* Python 3.11

## Special Instructions (Streamlit UI)

* Go to command prompt
* Locate 'streamlit_app.py' in same directory
* Type 'streamlit run streamlit_app.py'
